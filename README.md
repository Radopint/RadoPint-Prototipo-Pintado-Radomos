# RadoPint - Prototipo a escala para pintado de radomos aeronáuticos

Este repositorio contiene los archivos desarrollados para el trabajo de grado
"Prototipo a escala para el proceso de pintado de radomos aeronáuticos (RadoPint)".

## Contenido del repositorio

- `matlab/`: scripts para procesamiento del perfil del radomo, generación de trayectorias y conversión a G-code.
- `gcode/`: archivos G-code generados para pruebas en seco, calibración y aplicación de pintura.
- `esp32_sensores/`: código de la ESP32 para lectura del sensor VL53L0X y microswitches de seguridad.
- `marlin/`: configuración modificada de Marlin para el prototipo.
- `cad_planos/`: planos, archivos CAD, STL y STEP del sistema mecánico.
- `pruebas/`: protocolo de pruebas, datos experimentales y resultados de medición.
- `documentacion/`: manual de usuario y manual de desarrollador.

## Software requerido

- MATLAB
- Arduino IDE
- Marlin
- Pronterface
- SolidWorks o visor CAD compatible
